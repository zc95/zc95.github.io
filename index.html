nodejs搭建本地服务器环境 2018-01-11

Node.js® is a JavaScript runtime built on Chrome’s V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js’ package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Node.js® 是一个基于 Chrome V8 引擎的 JavaScript 运行时。 Node.js 使用高效、轻量级的事件驱动、非阻塞 I/O 模型。它的包生态系统，npm，是目前世界上最大的开源库生态系统。

安装nodejs

  1. 点击 https://nodejs.org 进入nodejs官网
  2. 有两个版本：LTS和Current，LTS是长期支持版本，成熟可靠
  3. next到底
  4. 打开终端，输入 node -v ，回车，如果出现类似这样 v6.11.4 的版本号就说明安装成功
